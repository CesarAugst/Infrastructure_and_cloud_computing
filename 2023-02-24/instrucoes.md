#Instrucoes
 * Ao executar o comando up ele cria dois containers e 1 network

##Como rodar
    * docker-compose up -d
        - executar na raiz do projeto

##Comandos úteis
    * estão rodando?
        - docker ps
    * estão se conectando?
        - docker network ls
    * não quero mais os containers
        - docker-compose down (na raiz do projeto)
